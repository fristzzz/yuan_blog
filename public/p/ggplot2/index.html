ggplot2

Mar 07, 2023
2 minute read

基本概念

library(ggplot2)
library(gapminder) # 数据集

gm <- gapminder

p <- ggplot(data = gm, mapping = aes(
    x = gdpPercap,
    y = lifeExp,
    color = continent,
    fill = continent
))

print(
    p +
        geom_point() +
        geom_smooth(method = "loess") +
        scale_x_log10(labels = scales::dollar)
)
## `geom_smooth()` using formula = 'y ~ x'

这个过程类似画一幅水彩画，有很多图层。

图形部件

一张统计图就是从数据到几何形状(geometic object, geom)所包含的图形属性(aesthetic attribute, aes)的一种映射

  1. data: 数据框 data.frame
  2. aes: 数据框中的数据变量映射到图形属性
    • x
    • y
    • color
    • size
    • shape
    • alpha
  3. geoms: 集合形状
    • geom_bar()
    • geom_density()
    • geom_freqpoly()
    • geom_histogram()
    • geom_violin()
    • geom_boxplot()
    • geom_col()
    • geom_point()
    • geom_smooth()
    • geom_tile()
    • geom_density2d()
    • geom_hex()
    • geom_count()
    • geom_text()
    • geom_sf()
  4. stats
  5. scales
  6. coord
  7. facet
  8. layer
  9. theme
  10. save

gapdata %>%
    ggplot(aes(x = continent)) +

    ggplot(aes(x = reorder(continent, continent, length))) +
    geom_bar()

gapdata %>%
    ggplot(aes(x = reorder(continent, continent, length))) +
    geom_bar() +
    coord_flip()

gapdata %>%

gapdata %>%
